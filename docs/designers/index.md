# Designer Track

Welcome. This track assumes **no programming background**. You'll work entirely with DataAssets in the editor's Details panel — no Blueprints, no C++.

## Start here

If you've never touched VNFramework before, work through these in order:

1. [Install the plugin](getting-started/installation.md) — get VNFramework into a project.
2. [Build your first project](getting-started/first-project.md) — create the top-level container.
3. [Build your first scene](getting-started/first-scene.md) — write dialogue, place a character, pick a background.
4. [Add your first choice](getting-started/first-choice.md) — branch the story.
